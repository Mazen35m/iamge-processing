# iamge-processing
This repository contains an image classification project using Google Teachable Machine.

Steps Taken:

    Trained an image recognition model using Teachable Machine to classify images into two categories (Dogs and Cats).

    Exported the trained model in TensorFlow Keras format (keras_model.h5 and labels.txt).

    Wrote a Python script in Google Colab to load the model, input a test image, and output the predicted class.
